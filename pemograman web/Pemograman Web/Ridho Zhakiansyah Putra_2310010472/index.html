About Me

Nama saya Ridho Zhakiansyah Putra, seorang mahasiswa Teknik Informatika yang penuh semangat dalam mengeksplorasi dunia teknologi. Saat ini, saya sedang menempuh semester 3, dan saya antusias untuk mempelajari hal-hal baru, terutama di bidang pengembangan web dan pemrograman. Saya menggunakan platform ini untuk berbagi proyek-proyek yang telah saya kerjakan, sebagai bagian dari perjalanan belajar saya di dunia IT.

Saya percaya bahwa teknologi memiliki kekuatan untuk mengubah dunia, dan saya bertekad untuk menjadi bagian dari perubahan itu. Dalam waktu luang, saya suka mencari tahu tren teknologi terbaru, mempelajari bahasa pemrograman baru, dan berkolaborasi dengan orang lain untuk menciptakan solusi kreatif. Terima kasih telah mengunjungi portofolio saya, dan jangan ragu untuk menghubungi saya jika Anda tertarik dengan karya saya atau ingin bekerja sama dalam proyek teknologi
